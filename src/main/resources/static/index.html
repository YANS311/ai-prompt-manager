📡 API 接口列表

GET /api/prompts

获取所有 Prompts（按创建时间倒序）。支持可选的 category 参数筛选。

请求示例: curl http://localhost:8080/api/prompts curl "http://localhost:8080/api/prompts?category=Coding" 响应示例: [{ "id": 1, "title": "Claude Code Agent", "content": "Expert software engineer", "category": "Coding", "createdAt": "2026-05-06T17:00:00", "updatedAt": "2026-05-06T17:00:00" }]
GET /api/prompts/{id}

根据 ID 获取单个 Prompt。

请求示例: curl http://localhost:8080/api/prompts/1 响应: 200 OK (成功) / 404 Not Found (不存在)
GET /api/prompts/search

根据关键词搜索标题（不区分大小写）。

请求示例: curl "http://localhost:8080/api/prompts/search?keyword=Claude"
POST /api/prompts

创建新的 Prompt。

请求示例: curl -X POST http://localhost:8080/api/prompts \ -H "Content-Type: application/json" \ -d '{ "title": "测试 Prompt", "content": "这是内容", "category": "Test" }' 响应: 201 Created
PUT /api/prompts/{id}

更新指定 ID 的 Prompt。

请求示例: curl -X PUT http://localhost:8080/api/prompts/1 \ -H "Content-Type: application/json" \ -d '{ "title": "更新后的标题", "content": "更新后的内容", "category": "Updated" }' 响应: 200 OK
DELETE /api/prompts/{id}

删除指定 ID 的 Prompt。

请求示例: curl -X DELETE http://localhost:8080/api/prompts/1 响应: 204 No Content

📚 数据模型

{ "id": Long, // 自增主键 "title": String, // 标题（必填） "content": String, // 内容 "category": String, // 分类 "createdAt": DateTime, // 创建时间（自动） "updatedAt": DateTime // 更新时间（自动） }
